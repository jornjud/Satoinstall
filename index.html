Hermes Agent Setup Guide

คู่มือการติดตั้งแบบ Step-by-Step

1. ติดตั้ง Ollama

เลือกระบบปฏิบัติการของคุณ:

irm https://ollama.com/install.ps1 | iex
curl -fsSL https://ollama.com/install.sh | sh

2. โหลด Model

เลือกโมเดลตาม VRAM ของการ์ดจอคุณ:

VRAM 16GB+ สำหรับ RTX 4070 / RTX 4080 / RTX 4090 / Apple M-series Pro/Max

โมเดลแนะนำจาก HauhauCS — ความสามารถสูงสุด รองรับ reasoning และ long context ได้ดี

ollama pull hf.co/HauhauCS/Qwen3.5-9B-Uncensored-HauhauCS-Aggressive:Q8_0
ollama pull hf.co/HauhauCS/Gemma-4-E4B-Uncensored-HauhauCS-Aggressive:Q8_K_P
VRAM 4GB สำหรับ RTX 3060 / RTX 3050 / การ์ดจอรุ่นเก่า / MacBook Air M1-M2

โมเดลแนะนำจาก HauhauCS — ขนาดเล็ก ใช้งานได้แม้ VRAM น้อย เหมาะกับเครื่องทั่วไป

ollama pull hf.co/HauhauCS/Qwen3.5-4B-Uncensored-HauhauCS-Aggressive:Q4_K_M
ollama pull hf.co/HauhauCS/Gemma-4-E2B-Uncensored-HauhauCS-Aggressive:IQ3_M

📊 เปรียบเทียบความต่าง

16GB — โมเดลใหญ่

  • ความสามารถ reasoning สูงกว่า
  • context length ยาวกว่า (32K+ tokens)
  • ตอบคำถามซับซ้อนได้ดีกว่า
  • ใช้ VRAM มาก — ไม่รองรับการ์ดจอรุ่นเก่า

4GB — โมเดลเล็ก

  • รองรับการ์ดจอทั่วไป (RTX 3060 ลงไป)
  • ใช้งานได้แม้ VRAM 4GB
  • ตอบคำถามทั่วไปได้ดี
  • context length สั้นกว่า (~8K tokens)

3. ติดตั้ง Hermes & Gateway

  1. ติดตั้งผ่านสคริปต์หลัก:
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
  2. เชื่อมต่อ Gateway สำหรับแอปแชท:
    hermes gateway setup

4. รัน Hermes Agent

หลังจากติดตั้งทุกอย่างเรียบร้อย รันคำสั่งด้านล่างเพื่อเปิดใช้งาน Hermes Agent:

Quick Start วิธีที่เร็วที่สุด
ollama launch hermes

🔧 สิ่งที่ ollama launch hermes ทำให้อัตโนมัติ

  1. ติดตั้ง — ถ้ายังไม่ได้ติดตั้ง Hermes จะ prompt ให้ติดตั้งผ่าน Nous Research install script
  2. เลือก Model — เลือก model ที่โหลดมาแล้ว (local หรือ cloud)
  3. Onboarding — ตั้งค่า Ollama provider ชี้ไปที่ http://127.0.0.1:11434/v1
  4. Gateway — เชื่อมต่อแพลตฟอร์มแชท (Telegram, Discord, Slack, WhatsApp, Signal, Email)

💬 เชื่อมต่อแพลตฟอร์มแชท

หลังติดตั้ง สามารถเชื่อมต่อ Telegram, Discord, Slack, WhatsApp, Signal หรือ Email ได้ด้วยคำสั่ง:

hermes gateway setup

⚠️ Windows

Hermes บน Windows ต้องใช้ WSL2 ก่อน ติดตั้งด้วยคำสั่ง:

wsl --install

จากนั้นรัน ollama launch hermes อีกครั้งจากใน WSL shell

⚠️ แก้ไขปัญหา (Troubleshooting)

หากพบ Error ว่า: Requires zstd for extraction ให้รันคำสั่งนี้เพื่อติดตั้งเครื่องมือที่จำเป็น:

sudo apt update && sudo apt install zstd -y

หมายเหตุ: คำสั่งนี้ใช้สำหรับสภาพแวดล้อม Linux หรือ WSL2 เท่านั้น
